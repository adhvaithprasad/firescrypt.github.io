What is firescrypt ?


Firescrypt in the most basic sense as said in the homepage is a library ,javascript to be specific as of this moment(we are still in dev mode). It is used to deploy web code (HTML) into embedded devices . I believe that by about year 2037 (random year which I generated by Math.random() method in javascript) almost all normal appliances will be controlled via something from anywhere in the world , in the present world internet is that something. So why not a way by which I could write the whole backend code of an microcontroller from a website ,spin it up rapidly and deploy it into the microcontroller almost instantly. The best part about writing code in HTML is that you could link a stylesheet from anywhere or source a js file ie; you don't even need to download it just one line of code for which you don't need to pay ,is fully open-source and be able to deploy a whole backend , lightweight code almost instantly into your microcontroller.
